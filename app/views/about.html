Software Security Service

Never do anything important unless you can validate what you are doing is what you are suppose to be doing.

The Critical Action Server uses keyless signature technology from Guardtime. Sign any file, URL, arbitary string.

Mathematically prove they are the same before you ever use them anywhere. No key vulnerabilities. Identity and time are secured in the signature and cannot be subjugated

  1. Choose a passphrase that only you know.
  2. Select the source you are protecting - a file, enter a URL or simple text
  3. Receive a .gtts signature file
  4. Communicate the source with the signature file and the passphrase, using secure means
  5. Whenever is required, 1 second, 1 minute, 10 years, resend the source, signature and passphrase
  6. If anything has been compromised validation will immediately fail
  7. If validation passes then you can gurarantee the source is the same as before and can be used with confidence (and indemnification)

If Target had implemented this technology they would never have been compromised or if they were they would be able to indemnify themselves and pass on appropriate ownership of compromise
